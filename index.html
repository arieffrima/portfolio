Individual Project

Web Applications
Development

BuyOnline is a simplified web-based online shopping system developed to allow customers to register, log in, and make purchases, as well as store managers to manage product listings and process sold items. This project demonstrates the use of HTML, JavaScript, PHP, XML, and AJAX for client-server communication and data management.

Files:

  • buyonline.htm: Main page with links to customer registration, customer login, and store manager login.
  • register.htm: Customer registration form where new customers can register with the system.
  • login.htm: Customer login page to authenticate customers and redirect to the buying page.
  • mlogin.htm: Store manager login page to authenticate managers and provide access to item listing and processing functions.
  • buying.htm: Customer buying page displaying available items and allowing customers to add items to the shopping cart.
  • listing.htm: Store manager page to add new items to the store inventory.
  • processing.htm: Store manager page to process sold items and update the inventory.
  • customer.xml: XML file that stores customer information such as customer ID, first name, surname, email, password, and phone number.
  • goods.xml: XML file that stores item information such as item number, name, description, price, quantity available, quantity on hold, and quantity sold.
  • manager.txt: A text file storing manager IDs and passwords.

Additional Information:

  • Uses session variables for tracking logins.
  • AJAX updates available items on the buying page.
  • Client-side and server-side input validation is implemented.

Notes: The system does not include payment processing or advanced password management, which may be added in future versions.

Programing

My Object-Oriented Programming project using Java and a Graphical User Interface (GUI), titled "GUI Parking Spot System." This project aims to enhance the original console-based interface by transitioning it into an intuitive and user-friendly GUI. Layout Management: The use of Layout Managers helped in organizing the GUI components in a way that is both functional and visually appealing. Importantly, this project is built from scratch without using any GUI development IDE or framework tools, demonstrating my proficiency in creating a fully functional GUI through pure coding techniques. In summary, this project showcases my ability to: Capture and respond to user events using ActionListeners. Design an easy-to-use GUI that effectively represents the system's functionality. Organize GUI components using Layout Managers and other essential elements.

Individual Project

Multi-Cloud Architecture Deployment
(AWS and Oracle)

In this comprehensive hybrid cloud architecture, I meticulously designed and implemented a robust integration between Oracle Cloud Infrastructure (OCI) and Amazon Web Services (AWS), showcasing my expertise in multi-cloud solutions. This architecture leverages a secure IPSec VPN tunnel to seamlessly connect a Virtual Cloud Network (VCN) in OCI with a Virtual Private Cloud (VPC) in AWS, ensuring secure, reliable, and efficient data transfer across cloud environments. The OCI segment includes public and private subnets that host critical components such as Bastion hosts, PHPAdmin, and a MySQL database, with stringent access controls managed through Network Security Groups (NSGs) and Security Lists. Meanwhile, the AWS segment incorporates a similar structure, with public and private subnets containing Bastion/Web servers and test instances, protected by Security Groups and Network ACLs (NACLs). This design not only enhances data security but also optimizes resource allocation, providing high availability and scalability. Key functionalities include the storage and synchronization of user-uploaded photos, with metadata shared seamlessly between OCI and AWS for further processing. This project underscores my ability to integrate complex cloud environments, ensuring robust security, efficient resource management, and seamless user experience.
