Programing

My Object-Oriented Programming project using Java and a Graphical User Interface (GUI), titled "GUI Parking Spot System." This project aims to enhance the original console-based interface by transitioning it into an intuitive and user-friendly GUI. I researched extensively to design a GUI that effectively represents parking spots, indicates when a spot is occupied by a car, and incorporates user-friendly components for input and selection. The system also displays all necessary messages within the GUI, ensuring clarity and ease of use. Key components of this project include: GUI Components: The interface is designed using various GUI components such as buttons, text fields, and labels to allow user interaction and input. Event Handling: I employed appropriate ActionListeners to capture user events and respond accordingly, ensuring the system's operations function as required. Layout Management: The use of Layout Managers helped in organizing the GUI components in a way that is both functional and visually appealing. Importantly, this project is built from scratch without using any GUI development IDE or framework tools, demonstrating my proficiency in creating a fully functional GUI through pure coding techniques. In summary, this project showcases my ability to: Capture and respond to user events using ActionListeners. Design an easy-to-use GUI that effectively represents the system's functionality. Organize GUI components using Layout Managers and other essential elements.

Individual Project

Multi-Cloud Architecture Deployment
(AWS and Oracle)

In this comprehensive hybrid cloud architecture, I meticulously designed and implemented a robust integration between Oracle Cloud Infrastructure (OCI) and Amazon Web Services (AWS), showcasing my expertise in multi-cloud solutions. This architecture leverages a secure IPSec VPN tunnel to seamlessly connect a Virtual Cloud Network (VCN) in OCI with a Virtual Private Cloud (VPC) in AWS, ensuring secure, reliable, and efficient data transfer across cloud environments. The OCI segment includes public and private subnets that host critical components such as Bastion hosts, PHPAdmin, and a MySQL database, with stringent access controls managed through Network Security Groups (NSGs) and Security Lists. Meanwhile, the AWS segment incorporates a similar structure, with public and private subnets containing Bastion/Web servers and test instances, protected by Security Groups and Network ACLs (NACLs). This design not only enhances data security but also optimizes resource allocation, providing high availability and scalability. Key functionalities include the storage and synchronization of user-uploaded photos, with metadata shared seamlessly between OCI and AWS for further processing. This project underscores my ability to integrate complex cloud environments, ensuring robust security, efficient resource management, and seamless user experience.
